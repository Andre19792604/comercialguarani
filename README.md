# Comercial Guarani — Protótipo Navegável

Versão de demonstração do site Comercial Guarani, hospedada via GitHub Pages.

## Sobre

Este é um **protótipo de design** (mock-up navegável) do e-commerce Comercial Guarani Ltda.

- Não processa pedidos reais
- Não conecta a sistemas de pagamento
- Não está integrado com ERP
- Serve para demonstração, validação visual e navegação de fluxos

## Telas incluídas

- `/` — Home v4 (versão aprovada com logo final)
- `/produto/` — Página de produto com galeria e abas técnicas
- `/categoria/` — Listagem de produtos com filtros laterais
- `/carrinho/` — Carrinho com cupom, frete e cálculo no Pix
- `/checkout/` — Checkout em 4 passos com tela de sucesso
- `/login/` — Cadastro e login com login social Google
- `/conta/` — Minha Conta com 9 abas (incluindo módulo B2B)
- `/empresas/` — Landing B2B com formulário de cadastro empresarial

## Modo Demo

Em qualquer tela, há um botão **"Modo Demo"** no canto inferior esquerdo que abre um painel lateral listando todas as 8 telas para navegação rápida.

## Tecnologia

HTML + CSS + JavaScript puros, sem frameworks.
